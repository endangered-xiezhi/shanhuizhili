## 项目背景

### 行业痛点
- **传统三会管理繁琐：** 股东大会、董事会、监事会的流程复杂，依赖人工操作，效率低下
- **合规风险高：** 缺乏实时合规审查机制，容易出现程序违规
- **文书工作量大：** 会议记录、决议文件等文书需要人工编写，耗时耗力
- **信息孤岛：** 法律法规更新不及时，难以确保会议决议符合最新法规要求

### 解决方案
通过 AI 技术实现三会管理的智能化、自动化，包括：
- 智能语音转写与实时合规审查
- AI 辅助文书生成
- 法律法规实时检索与风险预警
- 全流程数字化管理

## 产品定位

### 产品愿景
成为国内领先的企业三会治理智能化平台，帮助上市公司和企业集团实现三会运作的合规化、自动化、智能化。

### 产品使命
用 AI 技术赋能企业治理，降低合规风险，提升治理效率。

### 核心价值主张
- **智能合规：** 实时监控会议合规性，主动预警风险
- **高效便捷：** AI 自动生成文书，减少 80% 人工工作量
- **专业可靠：** 对标 2024 新《公司法》，确保合规性
- **数据驱动：** 沉淀治理数据，辅助决策分析

## 功能详情

### 1. 全局控制台 (Dashboard)

#### 1.1 会议概览
- **功能描述：** 展示所有三会会议的当前状态
- **数据展示：**
  - 待办会议数量
  - 进行中会议数量
  - 已结束会议数量
  - 近期会议日程列表
- **交互：** 点击会议卡片跳转到会议详情

#### 1.2 合规指数
- **功能描述：** 实时计算整体合规健康度
- **指标：**
  - 程序合规率（通知期限、出席比例等）
  - 文书合规率（格式、内容）
  - 风险预警数量
- **可视化：** 仪表盘展示，颜色区分风险等级

#### 1.3 风险预警
- **功能描述：** 实时预警潜在合规风险
- **风险类型：**
  - 程序性违规（通知期限不足、出席比例不够）
  - 实质性风险（关联交易、重大事项）
- **交互：** 点击风险项查看详情和处理建议

#### 1.4 法律库动态
- **功能描述：** 展示最新法律法规更新
- **内容：**
  - 新增法律法规
  - 法规修订通知
  - 相关监管问答

---

### 2. 会议管理 (Meeting Manager)

#### 2.1 创建会议
- **功能描述：** 创建新的三会会议
- **必填字段：**
  - 会议名称
  - 会议类型（股东会/董事会/监事会）
  - 会议日期
  - 会议地点
  - 议题列表
- **可选字段：**
  - 会议背景
  - 关联法律文件
  - 附件上传
- **自动化：**
  - 自动计算通知期限
  - 自动生成会议编号
  - 自动关联相关人员

#### 2.2 会议列表
- **功能描述：** 展示所有会议的列表视图
- **筛选条件：**
  - 按类型筛选
  - 按状态筛选（筹备中/进行中/已结束）
  - 按日期范围筛选
- **排序方式：**
  - 按日期排序
  - 按状态排序
  - 按合规指数排序
- **列表项信息：**
  - 会议名称、类型、日期
  - 合规指数
  - 通知状态
  - 操作按钮（查看/编辑/开始）

#### 2.3 会议日历
- **功能描述：** 以日历视图展示会议安排
- **视图类型：**
  - 月视图
  - 周视图
  - 日视图
- **交互：**
  - 点击日期创建会议
  - 点击会议查看详情
  - 拖拽调整会议时间

#### 2.4 会议详情
- **功能描述：** 查看会议的完整信息
- **信息展示：**
  - 基本信息（名称、类型、日期、地点）
  - 参会人员
  - 议题列表
  - 附件文件
  - 合规检查结果
  - 生成文书列表
- **操作：**
  - 编辑会议信息
  - 开始录音
  - 生成文书
  - 导出会议资料

### 3. 智能录音 (Recording Workspace)

#### 3.1 实时语音转写
- **功能描述：** 实时将会议语音转为文字
- **功能特性：**
  - 支持多角色识别（董事长、董事、监事等）
  - 实时显示转写结果
  - 自动添加时间戳
  - 支持录音回放与文本同步
- **技术支持：** 百度语音识别 API

#### 3.2 角色识别
- **功能描述：** 自动识别发言人身份
- **识别方式：**
  - 声纹识别（需要训练）
  - 语义分析
  - 人工标注辅助
- **角色类型：**
  - 董事长
  - 董事（含独立董事）
  - 监事
  - 高管
  - 其他

#### 3.3 AI 实时分析
- **功能描述：** AI 实时分析会议内容
- **分析维度：**
  - 议题识别
  - 决策提取
  - 风险识别
  - 法规匹配
- **展示方式：**
  - 侧边栏实时显示分析结果
  - 高亮显示风险内容
  - 关联相关法律条款

---

### 4. 合规审查 (Compliance Review)

#### 4.1 自动合规检查
- **功能描述：** 自动检查会议是否符合法律法规要求
- **检查项：**
  - 通知期限（股东会≥15日，董事会≥10日）
  - 出席比例（董事会≥1/2，股东会符合章程）
  - 表决权计算
  - 关联交易回避
  - 信息披露要求
- **检查依据：**
  - 2024 新《公司法》
  - 公司章程
  - 证监会相关规定
  - 自定义规则

#### 4.2 风险识别
- **功能描述：** 识别会议中的潜在风险点
- **风险类型：**
  - 程序性风险（流程违规）
  - 实质性风险（内容违规）
  - 信息披露风险
  - 关联交易风险
- **风险等级：**
  - Critical - 严重违规，必须立即处理
  - High - 高风险，建议修改
  - Medium - 中等风险，需要关注
  - Low - 低风险，建议优化

#### 4.3 合规报告生成
- **功能描述：** 自动生成合规审查报告
- **报告内容：**
  - 会议基本信息
  - 合规检查结果
  - 风险清单
  - 整改建议
  - 法律依据
- **输出格式：**
  - PDF
  - Word
  - HTML

## 用户体验
