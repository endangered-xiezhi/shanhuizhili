## 💡 快捷操作

### 创建新功能分支
```bash
git checkout -b feature/<feature-name>
```

### 提交代码
```bash
git add .
git commit -m "feat: <description>"
git push origin feature/<feature-name>
```

### 运行开发服务器
```bash
npm run dev
```

### 运行测试
```bash
npm run test
```

### 类型检查
```bash
npm run lint
```
